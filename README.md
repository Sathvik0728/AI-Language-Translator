# 🌍 AI Language Translator

An advanced AI-powered Language Translator built using Flask, JavaScript, HTML, and CSS.  
Translate text instantly between 50+ languages with voice input, speech output, dark mode, responsive UI, and modern AI-inspired design.

---

# 🚀 Live Demo

🔗 https://ai-language-translator-dvy5.onrender.com

# ✨ Features

✅ 50+ Languages Support  
✅ Real-Time Translation

# ⚡ Installation

## 1️⃣ Clone Repository

```bash
git clone https://github.com/Sathvik0728/AI-Language-Translator
```

## 2️⃣ Open Project Folder

```bash
cd AI-Language-Translator
```

## 3️⃣ Install Requirements

```bash
pip install -r requirements.txt
```

## 4️⃣ Run Application

```bash
python app.py
```
